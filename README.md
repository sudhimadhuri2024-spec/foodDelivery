# foodDelivery
Food Delivery Console Application built in Java using Segment Tree for efficient restaurant location search. Includes customer ordering system and admin menu management.

🍔 Food Delivery Application (Java)

This is a Java-based console food delivery system that simulates a simplified food ordering platform.
The application allows customers to search restaurants based on location and order food, while admins can manage restaurants and menu items.

🚀 Features

Customer Features

Search restaurants within a location radius

View restaurant menus

Add food items to order

View current order

Checkout and generate final bill

Admin Features

Add new restaurants

Add dishes to restaurant menus

View all available restaurants

🧠 Algorithms & Data Structures Used

Segment Tree

Used to efficiently search restaurants within a distance range

ArrayList

Used for storing restaurants, menus, and orders

Object-Oriented Programming

Classes like Restaurant, MenuItem, OrderItem

🏗 System Design

Main Components:

MenuItem → Represents food items

Restaurant → Stores restaurant information and menus

OrderItem → Represents items added to an order

RestaurantSegmentTree → Handles efficient location-based search

CustomerController → Customer interaction logic

ApplicationController → Admin and main system flow

FoodDeliveryApp → Main entry point of the application

📌 Technologies Used

Java

OOP Principles

Data Structures (Segment Tree)

Console-based UI
